## ขั้นตอนการติดตั้ง

1. **Clone โปรเจกต์จาก GitHub**

   ```bash
   git clone https://github.com/elephz/idotech-frontend.git
   cd idotech-frontend
   ```

2. **ติดตั้ง dependencies**

   ```bash
   npm install
   ```

3. **serve**

   ```bash
   npm run dev
   ```
